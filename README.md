# CYB333
CyberSecurity course work for CYB333
This script provides an intrusion detection system.
It is capable of detecting failed logins, port scans, SQL injections, brute force attacks, and privilege escalations.
This script can be located at https://github.com/najoshvy/CYB333/blob/main/Final
To run the code, open your Python editor. Then copy and paste the code into the Python editor. Make sure you save the file!

Prerequisites:
You must have a Python editor installed. If you do not, you can download it from https://www.python.org/downloads/ Download the most recent version that meets your operating system (OS). 
***When running the Python installation, the "Add python.exe to PATH" checkbox is selected!***
Once the install is complete, run Windows Command Prompt as administrator. Type python --version. If the Python version displays, Python has installed on your machine.

Running the Script:
A Python editor such as Visual Studio Code (VS Code) or PyCharm is required to modidy the script and/or run the script. These tools have built in checkers that will prompt you of an error. VS Code also has a built in artificial intelligence function that is capable of creating script or fixing errors.

Copy and paste the code into the Python editor. 
Run the script.
You should not receive any errors.

Results:
The results will be displayed in the terminal window.
